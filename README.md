# Global Superstore Sales Analysis & Forecasting

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze sales performance using the Global Superstore dataset. The dashboard focuses on key business metrics, time series trends, seasonality patterns, and short-term forecasting.

## Key Features
- Key Performance Indicators (Total Sales, Total Orders, Average Order Value)
- Sales trends over time with short-term forecasting
- Seasonal analysis by month and day of the week
- Regional and product category performance analysis
- Interactive slicers and tooltip insights

## Key Insights
- Sales show a consistent upward trend with noticeable seasonal fluctuations
- The West region contributes the highest sales performance
- Forecasting results indicate stable short-term sales behavior
